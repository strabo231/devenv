# Contributing to DevEnv

Thanks for your interest!

## Reporting Bugs
- Include OS and bash version
- Provide command and error
- Show `devenv status` output

## Suggesting Features
- Explain the use case
- Consider scope

## Pull Requests
1. Fork repository
2. Create feature branch
3. Test thoroughly
4. Follow code style
5. Update README
